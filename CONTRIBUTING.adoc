== Development

=== Useful development tools

The https://github.com/casey/just[just] command runner can be used. Run
`just --list` for more details.

.Run tests
[source,shell]
----
just test
----

.Run the formatter
[source,shell]
----
just fmt
----

.Run the linter
[source,shell]
----
just lint
----
